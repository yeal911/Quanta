## الميزات

### الميزات الأساسية

- **اختصار عالمي** - الافتراضي `Alt+R`، قابل للتخصيص في الإعدادات
- **بحث ضبابي** - مطابقة ذكية للكلمة المفتاحية / الاسم / الوصف، دعم المطابقة البادئية (اكتب `rec` للمطابقة مع `record`)
- **نتائج مجمعة** - النتائج منظمة حسب الفئة، مرتبة حسب درجة المطابقة داخل كل مجموعة
- **أوامر مخصصة** - يدعم أنواع Url و Program و Shell و Directory و Calculator
- **وضع المعامل** - اضغط `Tab` لإدخال المعامل (مثال: `g > rust` للبحث في Google)
- **Ctrl+رقم** - `Ctrl+1~9` لتنفيذ النتيجة رقم N فوراً
- **الإخفاء التلقائي** - يخفي تلقائياً عند النقر في مكان آخر

## الإدخال الذكي

لا حاجة لتحديد أمر. اكتب مباشرة في شريط البحث، سيتم التعرف على النتائج وعرضها في الوقت الفعلي.

### الحسابات الرياضية

أدخل تعبيرات رياضية مباشرة:

```
2+2*3         → 8
(100-32)/1.8  → 37.778
2^10          → 1024
15%4          → 3
-5+2          → -3
2^-3          → 0.125
2^3^2         → 512
```

العوامل المدعومة: `+ - * / % ^` مع دعم الأقواس والأرقام العشرية والإشارات الأحادية (مثل `-5` و`+3`).

الأسبقية: `^` (الأس) > `* / %` > `+ -`، وعملية الأس ترابطها يميني (مثال: `2^3^2 = 2^(3^2)`).

### تحويل الوحدات

التنسيق: `قيمة الوحدة_المصدر to الوحدة_الهدف`

```
100 km to miles    → 62.137 miles
100 c to f         → 212 °F
30 f to c          → -1.111 °C
1 kg to lbs        → 2.205 lbs
500 g to oz        → 17.637 oz
1 gallon to l      → 3.785 l
1 acre to m2       → 4046.856 m²
```

الفئات المدعومة: الطول، الوزن، درجة الحرارة، المساحة، الحجم

### تحويل العملات

التنسيق: `قيمة العملة_المصدر to العملة_الهدف` (يتطلب API Key في الإعدادات)

```
100 usd to cny     → ¥736.50 CNY
1000 cny to jpy    → ¥20,500 JPY
50 eur to gbp      → £42.30 GBP
```

دعم أكثر من 40 عملة. بيانات أسعار الصرف مخزنة محلياً (الافتراضي ساعة واحدة).

### تحويل الألوان

دعم تحويل HEX و RGB و HSL:

```
#E67E22            → rgb(230,126,34)  hsl(28,79%,52%)
rgb(230,126,34)    → #E67E22  hsl(28,79%,52%)
hsl(28,79,52%)     → #E67E22  rgb(230,126,34)
255,165,0          → #FFA500  hsl(38,100%,50%)
```

### أدوات النص

| البادئة | الوظيفة | المثال |
|---------|---------|--------|
| `base64 نص` | ترميز Base64 | `base64 hello` → `aGVsbG8=` |
| `base64d نص` | فك ترميز Base64 | `base64d aGVsbG8=` → `hello` |
| `md5 نص` | تجزئة MD5 | `md5 hello` → `5d41402a...` |
| `sha256 نص` | تجزئة SHA-256 | `sha256 hello` → `2cf24dba...` |
| `url نص` | ترميز/فك ترميز URL | `url hello world` → `hello%20world` |
| `json نصJSON` | تنسيق JSON | `json {"a":1}` →输出的格式 |

### البحث في Google

التنسيق: `g كلمة_مفتاحية`

```
g rust programming  → فتح بحث Google في المتصفح
```

### تنفيذ PowerShell

بادئة `>` لتنفيذ أوامر PowerShell مباشرة:

```
> Get-Process      → إظهار جميع العمليات
> dir C:\          → سرد ملفات C:\
```

عندما يتجاوز عدد الأحرف الحد (الافتراضي 20)، يتم إنشاء QR Code تلقائياً ونسخه إلى الحافظة:

```
https://github.com/yeal911/Quanta   → إنشاء تلقائي لـ QR Code
```

## ميزة التسجيل

أدخل `record` لإظهار لوحة التسجيل:

- **مصدر الصوت**: ميكروفون / مكبر صوت (صوت النظام) / ميكروفون + مكبر صوت
- **التنسيق**: m4a (AAC) / mp3
- **معدل البت**: 32 / 64 / 96 / 128 / 160 kbps
- **القنوات**: أحادي / ستيريو
- **العمليات**: بدء → إيقاف مؤقت / استئناف → إيقاف، محفوظ في المجلد المحدد
- **النقر بزر الماوس الأيمن على المعاملات** للتبديل

---

## سجل الحافظة

أدخل `clip` أو `clip كلمة_مفتاحية` للبحث في سجل الحافظة:

- تسجيل تلقائي للنص المنسوخ (بحد أقصى 50 إدخال)
- دعم تصفية الكلمات المفتاحية، يعرض آخر 20
- النقر على نتيجة للنسخ إلى الحافظة
- البيانات محفوظة بشكل دائم

## إدارة الأوامر

في الإعدادات (أمر `setting` أو النقر بزر الماوس الأيمن على الشريط)، علامة تبويب "إدارة الأوامر":

- تعديل الكلمات المفتاحية والأسماء والأنواع والمسارات مباشرة في الجدول
- دعم استيراد/تصدير JSON، عملي للنسخ الاحتياطي والهجرة

### أنواع الأوامر

| النوع | الوصف | المثال |
|-------|-------|--------|
| `Url` | فتح في المتصفح (يدعم `{param}`) | `https://google.com/search?q={param}` |
| `Program` | تشغيل برنامج | `notepad.exe` |
| `Shell` | تنفيذ في cmd | `ping {param}` |
| `Directory` | فتح مجلد في مستكشف الملفات | `C:\Users\Public` |
| `Calculator` | حساب تعبير | `{param}` |
