# دليل مستخدم Quanta

> النسخة العربية، وتمت مزامنتها مع دعم العمليات الرياضية الموسّع.

## البدء السريع
- `Alt+R` لإظهار/إخفاء Quanta
- اكتب مباشرة في مربع البحث

## الإدخال الذكي

### الحسابات الرياضية
أمثلة:
- `2+2*3` → `8`
- `(100-32)/1.8` → `37.778`
- `2^10` → `1024`
- `15%4` → `3`
- `-5+2` → `-3`
- `2^-3` → `0.125`
- `2^3^2` → `512`

المدعوم: `+ - * / % ^` مع الأقواس والكسور العشرية والإشارات الأحادية (`-5` و`+3`).

الأولوية: `^` > `* / %` > `+ -`، وعملية الأس يمينية الارتباط (`2^3^2 = 2^(3^2)`).

### إدخالات أخرى
- تحويل وحدات: `100 km to miles`
- تحويل عملات: `100 usd to cny`
- تحويل ألوان: HEX / RGB / HSL

## التسجيل
- المصدر: الميكروفون / مكبر الصوت / كلاهما
- الصيغة: `m4a` أو `mp3`

## سجل الحافظة
- يحتفظ بالعناصر المنسوخة مؤخرًا

## الإعدادات
- السمة، اللغة، بدء التشغيل التلقائي
- حد أقصى للنتائج
- مفتاح API لسعر الصرف ومدة التخزين المؤقت
