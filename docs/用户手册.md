# Quanta 用户手册

## 简介

Quanta 是一个 Windows 快速启动器，按 `Alt+R` 唤出搜索框，快速启动程序、执行命令。

---

## 快速开始

1. 运行程序 → 自动最小化到托盘
2. 按 `Alt+R` 唤出搜索框
3. 输入关键字搜索
4. `Enter` 执行，`Esc` 隐藏

---

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt+R` | 唤出/隐藏搜索框 |
| `Enter` | 执行选中命令 |
| `Tab` | 进入参数模式 |
| `Esc` | 隐藏窗口 |
| `↑` `↓` | 选择结果 |
| `Ctrl+1~9` | 快速执行第N个结果 |

## 特色功能

### 计算器
直接输入数学表达式：

```
2+2*3     → 8
sqrt(16)  → 4
100%3     → 1
```

### 单位转换
```
100 km to miles  → 62.137 miles
100 c to f      → 212 °F
1 kg to lbs     → 2.205 lbs
```

### 汇率转换
```
100 usd to cny   → 人民币金额
```
（需在设置中配置 API Key）

### 剪贴板历史
输入 `clip` 查看剪贴板历史

### 屏幕录音
输入 `record` 启动录音，支持麦克风/扬声器

### 二维码
输入超过 20 字符的文本自动生成二维码

---

## 参数模式

搜索到带参数的命令后，按 `Tab` 进入参数模式：

```
输入: g
按 Tab → g >
输入: g > rust → 执行搜索 "rust"
```

---

## 命令管理

右键托盘图标 → 设置 → 可添加/编辑/删除命令

### 命令类型

| 类型 | 说明 |
|------|------|
| Url | 打开网页 |
| Program | 启动程序 |
| Shell | 执行命令 |
| Directory | 打开文件夹 |
| Calculator | 计算表达式 |

---

## 设置

右键托盘图标 → 设置，可配置：
- 快捷键
- 主题（浅色/深色）
- 语言（中文/English/Español）
- 开机自启

右键托盘图标：
