# Quanta 用户手册

## 简介

Quanta 是一款 Windows 快速启动器，通过全局快捷键快速搜索并执行命令、启动程序。

---

## 快速开始

1. 运行程序 → 自动最小化到托盘
2. 按 `Alt+R` 唤出搜索框
3. 输入关键字搜索
4. `Enter` 执行，`Esc` 隐藏

---

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Alt+R` | 唤出/隐藏搜索框 |
| `Enter` | 执行选中命令 |
| `Tab` | 进入参数模式 |
| `Esc` | 隐藏窗口 |
| `↑` `↓` | 选择结果 |
| `Ctrl+1~9` | 快速执行 |

---

## 内置命令

| 关键字 | 功能 |
|--------|------|
| `cmd` | 命令提示符 |
| `powershell` | PowerShell |
| `notepad` | 记事本 |
| `calc` | 计算器 |
| `explorer` | 资源管理器 |
| `taskmgr` | 任务管理器 |
| `control` | 控制面板 |
| `regedit` | 注册表 |
| `lock` | 锁屏 |
| `shutdown` | 关机 |
| `restart` | 重启 |
| `sleep` | 睡眠 |
| `winrecord` | Windows 录音机 |

---

## 特色功能

### 剪贴板历史
- 输入 `clip` 查看剪贴板历史
- 支持搜索过滤

### 屏幕录音
- 输入 `record` 启动录音
- 支持麦克风/扬声器录制
- 可选 m4a/mp3 格式

### 二维码
- 输入超过 20 个字符的文本自动生成二维码
- 二维码自动复制到剪贴板

---

## 参数模式

搜索到带参数的命令后，按 `Tab` 进入参数模式：

```
输入: g
按 Tab → g >
输入: g > rust → 执行搜索 "rust"
```

---

## 命令管理

### 添加命令
1. 右键托盘 → 设置
2. 点击"添加"
3. 填写：关键字、名称、类型、路径
4. 点击"保存"

### 命令类型

| 类型 | 说明 | 示例 |
|------|------|------|
| Url | 打开网页 | `https://google.com/search?q={param}` |
| Program | 启动程序 | `notepad.exe` |
| Shell | 执行命令 | `ping {param}` |
| Directory | 打开文件夹 | `C:\Users` |
| Calculator | 计算表达式 | `{param}` |

### 导入/导出
- 点击"导入"/"导出"按钮
- JSON 格式，便于备份

---

## 设置

在设置界面可以配置：
- **语言**：中文/English
- **快捷键**：自定义全局热键
- **开机启动**：开机自动运行
- **最多显示**：搜索结果数量
- **长文本转二维码**：触发阈值

---

## 托盘菜单

右键托盘图标：
- 设置
- 语言 → 中文/English
- 关于
- 退出

---

## 主题
