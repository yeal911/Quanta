# Manual do Usuário do Quanta

> Versão em português sincronizada com o suporte matemático expandido.

## Início rápido
- `Alt+R` para mostrar/ocultar o Quanta
- Digite direto na barra de pesquisa

## Entrada inteligente

### Cálculos matemáticos
Exemplos:
- `2+2*3` → `8`
- `(100-32)/1.8` → `37.778`
- `2^10` → `1024`
- `15%4` → `3`
- `-5+2` → `-3`
- `2^-3` → `0.125`
- `2^3^2` → `512`

Suporte: `+ - * / % ^`, parênteses, decimais e sinais unários (`-5`, `+3`).

Precedência: `^` > `* / %` > `+ -`; potência é associativa à direita (`2^3^2 = 2^(3^2)`).

### Outras entradas
- Unidades: `100 km to miles`
- Moedas: `100 usd to cny`
- Cores: conversão HEX / RGB / HSL

## Gravação
- Fontes: microfone / sistema / ambos
- Formatos: `m4a`, `mp3`
