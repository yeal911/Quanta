# Quanta Benutzerhandbuch

> Diese Datei ist die deutsche Version des Benutzerhandbuchs und wurde mit den neuesten Rechenfunktionen synchronisiert.

## Schnellstart
- `Alt+R`: Quanta anzeigen/ausblenden
- Eingabe direkt im Suchfeld, Ergebnisse erscheinen sofort

## Intelligente Eingabe

### Mathematische Berechnung
Beispiele:
- `2+2*3` → `8`
- `(100-32)/1.8` → `37.778`
- `2^10` → `1024`
- `15%4` → `3`
- `-5+2` → `-3`
- `2^-3` → `0.125`
- `2^3^2` → `512`

Unterstützt: `+ - * / % ^`, Klammern, Dezimalzahlen, unäre Vorzeichen (`-5`, `+3`).

Priorität: `^` > `* / %` > `+ -`; Potenzen sind rechtsassoziativ (`2^3^2 = 2^(3^2)`).

### Weitere Eingaben
- Einheiten: `100 km to miles`
- Währungen: `100 usd to cny`
- Farben: HEX / RGB / HSL automatisch umrechnen

## Aufnahme
- Quellen: Mikrofon / Lautsprecher / Beide
- Format: `m4a` oder `mp3`
- Standard-Hotkey: `Ctrl+Shift+R`

## Zwischenablageverlauf
- Speichert zuletzt kopierte Inhalte
- Öffnen über Suchergebnis oder Shortcut

## Einstellungen
- Theme (Hell/Dunkel), Sprache, Autostart
- Max. Suchergebnisse
- QR-Schwelle
- Wechselkurs-API-Key und Cache-Dauer
