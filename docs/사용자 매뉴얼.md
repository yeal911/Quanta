# Quanta 사용자 매뉴얼

> 한국어 버전이며, 확장된 수학 연산 지원 내용을 반영했습니다.

## 빠른 시작
- `Alt+R`: Quanta 표시/숨김
- 검색창에 바로 입력

## 스마트 입력

### 수학 계산
예시:
- `2+2*3` → `8`
- `(100-32)/1.8` → `37.778`
- `2^10` → `1024`
- `15%4` → `3`
- `-5+2` → `-3`
- `2^-3` → `0.125`
- `2^3^2` → `512`

지원: `+ - * / % ^`, 괄호, 소수, 단항 부호(`-5`, `+3`).

우선순위: `^` > `* / %` > `+ -`; 거듭제곱은 우결합(`2^3^2 = 2^(3^2)`).

### 기타 입력
- 단위 변환: `100 km to miles`
- 환율 변환: `100 usd to cny`
- 색상 변환: HEX / RGB / HSL
