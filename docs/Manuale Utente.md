# Manuale Utente Quanta

> Versione italiana sincronizzata con i nuovi operatori matematici supportati.

## Avvio rapido
- `Alt+R` per mostrare/nascondere Quanta
- Digita direttamente nella barra di ricerca

## Input intelligente

### Calcoli matematici
Esempi:
- `2+2*3` → `8`
- `(100-32)/1.8` → `37.778`
- `2^10` → `1024`
- `15%4` → `3`
- `-5+2` → `-3`
- `2^-3` → `0.125`
- `2^3^2` → `512`

Supportati: `+ - * / % ^`, parentesi, decimali, segni unari (`-5`, `+3`).

Precedenza: `^` > `* / %` > `+ -`; la potenza è associativa a destra (`2^3^2 = 2^(3^2)`).

### Altri input
- Unità: `100 km to miles`
- Valute: `100 usd to cny`
- Colori: conversione HEX / RGB / HSL

## Registrazione
- Sorgenti: microfono / altoparlante / entrambi
- Formato: `m4a` / `mp3`

## Cronologia appunti
- Salva i contenuti copiati di recente

## Impostazioni
- Tema, lingua, avvio automatico
- Limite risultati
- API tassi di cambio + cache
