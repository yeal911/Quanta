# Manual de Usuario de Quanta

## Introducción

Quanta es un lanzador rápido para Windows. Presiona `Alt+R` para abrir el cuadro de búsqueda y lanzar programas o ejecutar comandos rápidamente.

---

## Inicio Rápido

1. Ejecuta el programa → se minimiza a la bandeja
2. Presiona `Alt+R` para abrir el cuadro de búsqueda
3. Escribe una palabra clave para buscar
4. Presiona `Enter` para ejecutar, `Esc` para ocultar

---

## Atajos de Teclado

| Atajo | Acción |
|-------|--------|
| `Alt+R` | Mostrar/ocultar cuadro de búsqueda |
| `Enter` | Ejecutar comando seleccionado |
| `Tab` | Entrar en modo de parámetros |
| `Esc` | Ocultar ventana |
| `↑` `↓` | Navegar resultados |
| `Ctrl+1~9` | Ejecutar resultado N directamente |

## Características Especiales

### Calculadora
Escribe expresiones matemáticas directamente:

```
2+2*3     → 8
sqrt(16)  → 4
100%3     → 1
```

### Conversión de Unidades
```
100 km to miles  → 62.137 miles
100 c to f      → 212 °F
1 kg to lbs     → 2.205 lbs
```

### Conversión de Moneda
```
100 usd to cny   → cantidad en yuanes
```
(Requiere API Key en configuración)

### Historial del Portapapeles
Escribe `clip` para ver el historial

### Grabación de Pantalla
Escribe `record` para iniciar grabación

### Código QR
Texto de más de 20 caracteres genera código QR automáticamente

---

## Modo de Parámetros

Después de coincidir con un comando con parámetros, presiona `Tab`:

```
Escribe: g
Presiona Tab → g >
Escribe: g > rust → ejecuta búsqueda "rust"
```

---

## Gestión de Comandos

Clic derecho en icono de bandeja → Configuración → Agregar/Editar/Eliminar comandos

### Tipos de Comandos

| Tipo | Descripción |
|------|-------------|
| Url | Abrir página web |
| Program | Iniciar programa |
| Shell | Ejecutar comando |
| Directory | Abrir carpeta |
| Calculator | Calcular expresión |

---

## Configuración

Clic derecho en icono de bandeja → Configuración:
- Tecla rápida
- Tema (Claro/Oscuro)
- Idioma (中文/English/Español)
- Inicio automático

---

## Menú de Bandeja

Clic derecho en el icono:
- Configuración
- Idioma
- Acerca de
- Salir
