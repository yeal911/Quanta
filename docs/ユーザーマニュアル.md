# Quanta ユーザーマニュアル

> 日本語版。数学演算サポート拡張に合わせて更新済みです。

## クイックスタート
- `Alt+R` で Quanta の表示/非表示
- 検索ボックスへ直接入力

## スマート入力

### 数学計算
例:
- `2+2*3` → `8`
- `(100-32)/1.8` → `37.778`
- `2^10` → `1024`
- `15%4` → `3`
- `-5+2` → `-3`
- `2^-3` → `0.125`
- `2^3^2` → `512`

対応: `+ - * / % ^`、括弧、小数、単項符号（`-5`、`+3`）。

優先順位: `^` > `* / %` > `+ -`。べき乗は右結合（`2^3^2 = 2^(3^2)`）。

### その他
- 単位変換: `100 km to miles`
- 為替変換: `100 usd to cny`
- 色変換: HEX / RGB / HSL

## 録音
- 入力元: マイク / スピーカー / 両方
- 形式: `m4a` / `mp3`

## クリップボード履歴
- 最近コピーした内容を保存

## 設定
- テーマ、言語、自動起動
- 最大結果数
- 為替 API キーとキャッシュ
