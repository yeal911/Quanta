# Manuel Utilisateur Quanta

> Version française synchronisée avec les nouvelles capacités de calcul mathématique.

## Démarrage rapide
- `Alt+R` : afficher/masquer Quanta
- Saisir directement dans la barre de recherche

## Saisie intelligente

### Calculs mathématiques
Exemples :
- `2+2*3` → `8`
- `(100-32)/1.8` → `37.778`
- `2^10` → `1024`
- `15%4` → `3`
- `-5+2` → `-3`
- `2^-3` → `0.125`
- `2^3^2` → `512`

Prise en charge : `+ - * / % ^`, parenthèses, décimales, signes unaires (`-5`, `+3`).

Priorité : `^` > `* / %` > `+ -` ; la puissance est associative à droite (`2^3^2 = 2^(3^2)`).

### Autres entrées
- Unités : `100 km to miles`
- Devises : `100 usd to cny`
- Couleurs : conversion HEX / RGB / HSL

## Enregistrement
- Sources : micro / haut-parleur / les deux
- Formats : `m4a`, `mp3`
- Raccourci par défaut : `Ctrl+Shift+R`
