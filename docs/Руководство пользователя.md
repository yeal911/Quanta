# Руководство пользователя Quanta

> Русская версия синхронизирована с расширенной поддержкой математических операций.

## Быстрый старт
- `Alt+R` — показать/скрыть Quanta
- Вводите запрос прямо в строку поиска

## Умный ввод

### Математические вычисления
Примеры:
- `2+2*3` → `8`
- `(100-32)/1.8` → `37.778`
- `2^10` → `1024`
- `15%4` → `3`
- `-5+2` → `-3`
- `2^-3` → `0.125`
- `2^3^2` → `512`

Поддержка: `+ - * / % ^`, скобки, десятичные числа и унарные знаки (`-5`, `+3`).

Приоритет: `^` > `* / %` > `+ -`; степень правоассоциативна (`2^3^2 = 2^(3^2)`).

### Другие форматы ввода
- Единицы: `100 km to miles`
- Валюты: `100 usd to cny`
- Цвета: HEX / RGB / HSL

## Запись
- Источник: микрофон / динамик / оба
- Формат: `m4a`, `mp3`

## История буфера обмена
- Хранит недавно скопированные элементы
