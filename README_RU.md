## Функции

### Основные функции

- **Глобальная горячая клавиша** - По умолчанию `Alt+R`, настраивается в параметрах
- **Нечеткий поиск** - Интеллектуальное сопоставление по ключевому слову / имени / описанию, поддержка префиксного нечеткого поиска (введите `rec` для соответствия `record`)
- **Группировка результатов** - Результаты организованы по категориям, отсортированы по оценке соответствия внутри каждой группы
- **Пользовательские команды** - Поддержка типов Url, Program, Shell, Directory, Calculator
- **Режим параметров** - Нажмите `Tab` для ввода параметра (например, `g > rust` для поиска в Google)
- **Ctrl+Цифра** - `Ctrl+1~9` для мгновенного выполнения N-го результата
- **Автоскрытие** - Автоматически скрывается при щелчке в другом месте

## Умный ввод

Нет необходимости выбирать команду. Введите прямо в строку поиска, результаты распознаются и отображаются в реальном времени.

### Математические вычисления

Вводите математические выражения напрямую:

```
2+2*3         → 8
(100-32)/1.8  → 37.778
2^10          → 1024
15%4          → 3
```

Поддерживаемые операторы: `+ - * / % ^`, поддержка скобок и десятичных дробей.

### Конвертация единиц

Формат: `значение исходная_единица в целевую_единицу`

```
100 km to miles    → 62.137 miles
100 c to f         → 212 °F
30 f to c          → -1.111 °C
1 kg to lbs        → 2.205 lbs
500 g to oz        → 17.637 oz
1 gallon to l      → 3.785 l
1 acre to m2       → 4046.856 m²
```

Поддерживаемые категории: длина, вес, температура, площадь, объем

### Конвертация валют

Формат: `значение исходная_валюта в целевую_валюту` (требуется API Key в настройках)

```
100 usd to cny     → ¥736.50 CNY
1000 cny to jpy    → ¥20,500 JPY
50 eur to gbp      → £42.30 GBP
```

Поддержка более 40 валют. Данные о курсах кэшируются локально (по умолчанию 1 час).

### Конвертация цветов

Поддержка конвертации HEX, RGB, HSL:

```
#E67E22            → rgb(230,126,34)  hsl(28,79%,52%)
rgb(230,126,34)    → #E67E22  hsl(28,79%,52%)
hsl(28,79,52%)     → #E67E22  rgb(230,126,34)
255,165,0          → #FFA500  hsl(38,100%,50%)
```

### Текстовые инструменты

| Префикс | Функция | Пример |
|---------|---------|--------|
| `base64 текст` | Кодировка Base64 | `base64 hello` → `aGVsbG8=` |
| `base64d текст` | Декодировка Base64 | `base64d aGVsbG8=` → `hello` |
| `md5 текст` | Хэш MD5 | `md5 hello` → `5d41402a...` |
| `sha256 текст` | Хэш SHA-256 | `sha256 hello` → `2cf24dba...` |
| `url текст` | Кодировка/Декодировка URL | `url hello world` → `hello%20world` |
| `json JSONстрока` | Форматирование JSON | `json {"a":1}` → форматированный вывод |

### Поиск Google

Формат: `g ключевое_слово`

```
g rust programming  → Открыть поиск Google в браузере
```

### Выполнение PowerShell

Префикс `>` для выполнения команд PowerShell напрямую:

```
> Get-Process      → Показать все процессы
> dir C:\          → Список файлов C:\
```

### Генерация QR-кода

Когда количество символов превышает порог (по умолчанию 20), автоматически генерируется QR-код и копируется в буфер обмена:

```
https://github.com/yeal911/Quanta   → Автоматическая генерация QR-кода
```

## Функция записи

Введите `record` для отображения панели записи:

- **Источник звука**: Микрофон / Динамик (системное аудио) / Микрофон + Динамик
- **Формат**: m4a (AAC) / mp3
- **Битрейт**: 32 / 64 / 96 / 128 / 160 kbps
- **Каналы**: Моно / Стерео
- **Операции**: Старт → Пауза/Продолжить → Стоп, сохранение в указанную директорию
- **Щелчок правой кнопкой по параметрам** для переключения

---

## История буфера обмена

Введите `clip` или `clip ключевое_слово` для поиска в истории буфера обмена:

- Автоматическая запись скопированного текста (максимум 50 записей)
- Поддержка фильтрации по ключевому слову, показывает последние 20
- Щелчок по результату копирует в буфер обмена
- Данные сохраняются постоянно

## Интерфейс и опыт

- **Светлая/Темная тема** - Переключение в настройках, автоматическое сохранение
- **Мультиязычность** - 中文 / English / Español / 日本語 / 한국어 / Français / Deutsch / Português / Русский / Italiano / العربية, мгновенное переключение
- **Системный трей** - Резидент в трее, контекстное меню для быстрых действий
- **Автозапуск** - Активация одним щелчком в настройках
- **Toast-уведомления** - Успех/ошибка операций сообщается мгновенно

## Быстрый старт

### Системные требования

- Windows 10 / 11 (x64)
- [.NET 8.0 Runtime](https://dotnet.microsoft.com/download/dotnet/8.0)

```bash
dotnet run
```

```bash
dotnet publish -c Release -r win-x64 --self-contained false
```

## Управление командами

В настройках (команда `setting` или щелчок правой кнопкой по трею), вкладка "Управление командами":

- Редактируйте ключевые слова, имена, типы и пути непосредственно в таблице
- Поддержка импорта/экспорта JSON, удобно для резервного копирования и миграции

### Типы команд

| Тип | Описание | Пример |
|-----|----------|--------|
| `Url` | Открыть в браузере (поддерживает `{param}`) | `https://google.com/search?q={param}` |
| `Program` | Запустить программу | `notepad.exe` |
| `Shell` | Выполнить в cmd | `ping {param}` |
| `Directory` | Открыть папку в проводнике | `C:\Users\Public` |
| `Calculator` | Вычислить выражение | `{param}` |

## Техническая архитектура

```
├── App.xaml / App.xaml.cs          # Точка входа, трей, тема, жизненный цикл

│   │   ├── ApplicationSearchProvider.cs
│   │   ├── FileSearchProvider.cs

├── Core/
│   ├── Config/                      # AppConfig, ConfigLoader
```
